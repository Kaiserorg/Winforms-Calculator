# WinForms Calculator

A simple calculator built in C# using Windows Forms. This was my first C# project, made to get familiar with WinForms and basic event handling.

## Features
- Addition, subtraction, multiplication, division
- Expression evaluation via `DataTable.Compute()`

## Built With
- C# / .NET
- Windows Forms

## How to Run
1. Clone the repo
2. Open `WinFormsApp1.csproj` in Visual Studio
3. Hit F5
